## 如何深度使用

目前支持的設置選項如下

- 快捷鍵 F4
- 開機啟動 是/否
- 背景色 #525252
- 透明度 0-100
- 字體顏色 #ffffff
- 裝飾色 #1e1e1e
- 基準大小 14
- 顯示欄數 4
- 視窗高度 750
- 視窗寬度 1200
